A simple, clean, and responsive Weather App that fetches current weather and a 7-day forecast for any city using the OpenWeatherMap API. It also tries to detect your location automatically for quick weather updates.

---

## Features

- **Current Weather**: Displays temperature, weather description, humidity, wind speed, and date/time.
- **7-Day Forecast Chart**: Interactive line chart showing temperature and humidity trends over the week.
- **Geolocation Support**: Automatically fetches weather data based on your current location (with permission).
- **Search by City**: Search for any city worldwide to get weather updates.
- **Responsive Design**: Works well on both desktop and mobile devices.
